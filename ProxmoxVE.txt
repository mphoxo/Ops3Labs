#Login into Proxmox
	Enter address of Proxmox(Debian) in firefox(https://<ip>:8006)
	User: root

#Deploy VM
	1. Click "Create VM" (Top Right)
	2. Set up VM based on requirements
	3. Confirm & Finish
	4. You should see "QM CREATE - Status: OK." at bottom

#Taking snapshort of empty state
	>Select VM 100.
	>Go to Snapshots.
	>Click Take Snapshot.

#Booting VM
	>Click Console -> Start Now.
	>Watch the VM boot

#Replicating VM(Clone)
	 >Right-click VM 100, select Clone.
